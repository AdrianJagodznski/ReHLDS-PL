# Repozytorium ReHLDS
Zawiera najnowsze wersje ReHLDS oraz dodatków związanych z nim

### Instalacja 
1. Pobierz paczkę
2. Zaloguj się do FTP Twojego serwera
3. Wrzuć pliki zgodnie z ich strukturą.
Paczka zawiera zawsze najnowsze dostępne wersje plików.
